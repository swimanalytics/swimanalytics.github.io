FAQ

Pertanyaan Umum

Bagaimana SwimAnalytics mendapatkan data renang saya?

SwimAnalytics disinkronkan dengan Apple Health untuk mengimpor latihan renang yang dicatat oleh perangkat atau aplikasi yang kompatibel. Ini termasuk jam tangan pintar, pelacak kebugaran, dan entri manual. Aplikasi memproses data ini secara lokal untuk menghitung metrik lanjutan.

Apa itu tes CSS dan bagaimana cara melakukannya?

CSS (Critical Swim Speed) adalah protokol ilmiah yang menggunakan 2 renang dengan usaha maksimal: 400m dan 200m dengan istirahat 10-20 menit di antaranya. Aplikasi menghitung ambang aerobik Anda dari waktu ini dan secara otomatis menyesuaikan semua zona latihan. Ulangi setiap 6-8 minggu untuk melacak kemajuan.

Apakah data saya diunggah ke cloud?

Tidak. SwimAnalytics memproses semua data secara lokal di iPhone Anda. Tidak ada server eksternal, tanpa akun cloud, tanpa transfer data. Anda mengontrol ekspor: buat file JSON, CSV, HTML atau PDF dan bagikan sesuai keinginan Anda.

Bisakah saya menggunakan SwimAnalytics untuk renang air terbuka?

Ya. SwimAnalytics berfungsi dengan latihan renang apa pun di Apple Health, termasuk air terbuka. Aplikasi beradaptasi dengan metrik yang tersedia baik di kolam maupun air terbuka, memberikan analitik yang relevan untuk setiap lingkungan.

Apa perbedaan antara paket bulanan dan tahunan?

Kedua paket menawarkan fitur yang identik: semua metrik, zona tak terbatas, perbandingan temporal, ekspor ganda dan pembaruan gratis. Satu-satunya perbedaan adalah harga: tahunan menghemat 18% (setara dengan ‚Ç¨3.25/bulan vs ‚Ç¨3.99/bulan).

Bisakah saya membatalkan langganan kapan saja?

Ya. Langganan dikelola melalui App Store, jadi Anda dapat membatalkan kapan saja dari Pengaturan ‚Üí [Nama Anda] ‚Üí Langganan. Jika Anda membatalkan, Anda akan tetap memiliki akses hingga akhir periode penagihan Anda saat ini.

Siap Mengubah Renang Anda?

Bergabunglah dengan ribuan perenang yang menggunakan metrik ilmiah untuk meningkatkan performa mereka. Mulai uji coba gratis 7 hari Anda hari ini.

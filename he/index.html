מדדים מדעיים

מהירות שחייה קריטית (CSS) קובעת את סף האירובי שלך, ומאפשרת לחשב את ציון עומס האימון (TSS) ומעקב ביצועים עם CTL/ATL/TSB המבוסס על מחקר מדעי מוכח בספורט.

אזורי אימון

7 אזורי אימון מותאמים אישית המכוילים לפי ה-CSS שלך. ייעל כל אימון לפיתוח התאוששות, אירובי, סף או VO₂max.

השוואות חכמות

השוואות תקופתיות שבועיות, חודשיות ושנתיות עם זיהוי אוטומטי של מגמות ושינויים באחוזים עבור כל המדדים.

פרטיות מוחלטת

כל הנתונים מעובדים מקומית במכשיר שלך. ללא שרתים, ללא ענן, ללא מעקב. אתה הבעלים ושולט בנתוני השחייה שלך.

ייצוא לכל מקום

ייצא אימונים ואנליטיקה בפורמטים JSON, CSV, HTML או PDF. תואם למאמנים, גיליונות אלקטרוניים ופלטפורמות אימון.

ביצועים מיידיים

הפעלת האפליקציה בפחות מ-0.35 שניות עם ארכיטקטורת local-first. הצג את האימונים שלך באופן מיידי ללא המתנה לסנכרונים או הורדות.

שאלות נפוצות

שאלות נפוצות

איך SwimAnalytics מקבל את נתוני השחייה שלי?

SwimAnalytics מסתנכרן עם Apple Health כדי לייבא אימוני שחייה שנרשמו על ידי כל מכשיר או אפליקציה תואמת. זה כולל שעונים חכמים, מעקבי כושר והזנות ידניות. האפליקציה מעבדת את הנתונים הללו מקומית כדי לחשב מדדים מתקדמים.

מה זה בדיקת CSS ואיך מבצעים אותה?

CSS (Critical Swim Speed) הוא פרוטוקול מדעי שמשתמש ב-2 שחיות במאמץ מקסימלי: 400 מ' ו-200 מ' עם 10-20 דקות מנוחה ביניהן. האפליקציה מחשבת את הסף האירובי שלך מהזמנים האלה ומכיילה אוטומטית את כל אזורי האימון. חזור כל 6-8 שבועות כדי לעקוב אחר ההתקדמות.

הנתונים שלי עולים לענן?

לא. SwimAnalytics מעבד את כל הנתונים מקומית ב-iPhone שלך. אין שרתים חיצוניים, אין חשבונות בענן, אין העברות נתונים. אתה שולט בייצואים: צור קבצי JSON, CSV, HTML או PDF ושתף אותם כרצונך.

האם אני יכול להשתמש ב-SwimAnalytics לשחייה במים פתוחים?

כן. SwimAnalytics עובד עם כל אימון שחייה ב-Apple Health, כולל מים פתוחים. האפליקציה מסתגלת למדדים הזמינים הן בבריכה והן במים פתוחים, ומספקת ניתוח רלוונטי לכל סביבה.

מה ההבדל בין התכניות החודשית והשנתית?

שתי התכניות מציעות תכונות זהות: כל המדדים, אזורים ללא הגבלה, השוואות זמן, ייצואים מרובים ועדכונים חינם. ההבדל היחיד הוא המחיר: השנתי חוסך 18% (שווה ערך ל-€3.25/חודש לעומת €3.99/חודש).

האם אני יכול לבטל את המנוי שלי בכל עת?

כן. המנויים מנוהלים דרך App Store, כך שאתה יכול לבטל בכל עת מהגדרות → [השם שלך] → מנויים. אם אתה מבטל, תשמור על גישה עד סוף תקופת החיוב הנוכחית שלך.

מוכן לשנות את השחייה שלך?

הצטרף לאלפי שחיינים שמשתמשים במדדים מדעיים כדי לשפר את הביצועים שלהם. התחל את הניסיון החינמי ל-7 ימים שלך היום.

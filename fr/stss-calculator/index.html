Calculatrice Gratuita de TSS para Natation

Calcula el Training Stress Score para entraînements de natation - La única calculatrice gratuita de sTSS

¿Qué es el TSS de Natation (sTSS)?

Swimming Training Stress Score (sTSS) cuantifica la charge de entraînement de una sesión de natation combinando intensidad y duración. Está adaptado de la metodología TSS del ciclismo, utilizando tu Critical Swim Speed (CSS) como allure seuil. Un entraînement de 1 hora a allure CSS = 100 sTSS.

Interpretación de la Charge de Entraînement

Cómo se Calcula el sTSS

Fórmula

sTSS = (Duración en horas) × (Factor de Intensidad)² × 100

Donde:

  • Factor de Intensidad (IF) = Allure CSS / Allure Promedio del Entraînement
  • Duración = Tiempo total del entraînement en horas
  • Allure CSS = Tu allure seuil del test CSS

Ejemplo Práctico

Detalles del Entraînement:

  • Allure CSS: 1:49/100m (109 segundos)
  • Duración del Entraînement: 60 minutos (1 hora)
  • Allure Promedio: 2:05/100m (125 segundos)

Paso 1: Calcular el Factor de Intensidad

IF = Allure CSS / Allure del Entraînement
IF = 109 / 125
IF = 0.872

Paso 2: Calcular el sTSS

sTSS = 1.0 horas × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Interpretación: Este entraînement de 60 minutos a allure suave (más lento que CSS) generó 76 sTSS - una charge de entraînement moderada adecuada para construir la base aeróbica.

Interpretando los Valores de sTSS

Rango sTSS Charge de Entraînement Tiempo de Récupération Ejemplo de Entraînement
< 50 Baja Mismo día Nado suave de 30 min, ejercicios técnicos
50-100 Moderada 1 día Resistencia de 60 min, allure constante
100-200 Alta 1-2 días Series de seuil de 90 min, intervalos a allure de competición
200-300 Muy Alta 2-3 días Entraînement duro de 2 horas, múltiples bloques de seuil
> 300 Extrema 3+ días Competición larga (>2 horas), ultra-resistencia

Guía de sTSS Semanal

El sTSS semanal objetivo depende de tu nivel de entraînement y objetivos:

Nadadores Recreativos

sTSS Semanal: 150-300

2-3 entraînements por semana, 50-100 sTSS cada uno. Enfoque en technique y construcción de base aeróbica.

Nadadores de Fitness / Triatletas

sTSS Semanal: 300-500

3-4 entraînements por semana, 75-125 sTSS cada uno. Mezcla de resistencia aeróbica y trabajo de seuil.

Nadadores Másters Competitivos

sTSS Semanal: 500-800

4-6 entraînements por semana, 80-150 sTSS cada uno. Entraînement estructurado con periodización.

Nadadores Elite / Universitarios

sTSS Semanal: 800-1200+

8-12 entraînements por semana, sesiones dobles. Alto volumen con gestión de récupération crítica.

⚠️ Notas Importantes

  • Requiere CSS preciso: Tu CSS debe estar actualizado (testeado dentro de 6-8 semanas) para un sTSS preciso.
  • Calcul simplificado: Esta calculatrice usa allure promedio. El sTSS avanzado usa Normalized Graded Pace (NGP) que tiene en cuenta la estructura de intervalos.
  • No para trabajo técnico: El sTSS solo mide el estrés físico del entraînement, no el desarrollo de habilidades.
  • Variación individual: El mismo sTSS se siente diferente para diferentes nageurs. Ajusta las guías según tu récupération.

Por Qué Importa el sTSS

El Training Stress Score es la base para:

  • CTL (Chronic Training Load): Tu nivel de forme physique - promedio ponderado exponencialmente de 42 días del sTSS diario
  • ATL (Acute Training Load): Tu fatigue - promedio ponderado exponencialmente de 7 días del sTSS diario
  • TSB (Training Stress Balance): Tu estado de forma - TSB = CTL - ATL (positivo = fresco, negativo = fatigado)
  • Periodización: Planificar fases de entraînement (base, desarrollo, pico, descenso) usando progresiones de CTL objetivo
  • Gestión de Récupération: Saber cuándo empujar y cuándo descansar según el TSB

Consejo Pro: Sigue tu CTL

Registra el sTSS diario en una hoja de calcul o diario de entraînement. Calcula tu promedio de 42 días (CTL) semanalmente. Objetivo de aumento de 5-10 puntos CTL por semana durante la construcción de base. Mantén o reduce ligeramente el CTL durante el descenso (1-2 semanas antes de la competición).

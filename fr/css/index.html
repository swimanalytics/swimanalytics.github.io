Vitesse Crítica de Nado (CSS)

La Base del Entraînement de Natation Basado en Datos

¿Qué es la Vitesse Crítica de Nado (CSS)?

La Vitesse Crítica de Nado (CSS) es la vitesse máxima teórica de natation que puedes mantener sin agotarte. Representa tu allure de seuil aeróbico, típicamente correspondiente a 4 mmol/L de lactato en sangre y sostenible durante aproximadamente 30 minutos. El CSS se calcula utilizando una test contrarreloj de 400m y 200m para determinar zones de entraînement personalizadas.

La Vitesse Crítica de Nado (CSS) representa la vitesse máxima teórica de natation que puedes mantener continuamente sin agotarte. Es tu seuil aeróbico en el agua—la intensidad donde la producción de lactato iguala la eliminación de lactato.

🎯 Significado Fisiológico

El CSS se corresponde estrechamente con:

  • Seuil de Lactato 2 (LT2) - Segundo seuil ventilatorio
  • Estado Máximo Estable de Lactato (MLSS) - Nivel más alto de lactato sostenible
  • Allure de Seuil Funcional (FTP) - Equivalente en natation al FTP de ciclismo
  • ~4 mmol/L de lactato en sangre - Marcador tradicional de OBLA

Por Qué Importa el CSS

El CSS es la métrica fundamental que desbloquea todo el análisis avanzado de charge de entraînement:

  • Zones de Entraînement: Personaliza las zones de intensidad según tu fisiología
  • Calcul de sTSS: Permite la cuantificación precisa del Índice de Estrés de Entraînement
  • CTL/ATL/TSB: Requerido para las métricas del Gráfico de Gestión del Performance
  • Seguimiento del Progreso: Medida objetiva de la mejora de la condición aeróbica
⚠️ Dependencia Crítica: Sin una test de CSS válida, las métricas avanzadas de charge de entraînement (sTSS, CTL, ATL, TSB) no se pueden calcular. Un CSS inexacto corromperá todo el análisis de entraînement posterior.

Protocolo de Test CSS

📋 Protocolo Estándar

  1. Calentamiento

    300-800m de natation suave, ejercicios técnicos y aceleraciones progresivas para prepararse para el esfuerzo máximo.

  2. Test Contrarreloj de 400m

    Esfuerzo máximo sostenido desde salida de pared (sin clavado). Registra el tiempo al segundo. Objetivo: 400m sostenibles más rápidos.

  3. Récupération Completa

    5-10 minutos de natation suave o descanso completo. Esto es CRÍTICO para resultados precisos.

  4. Test Contrarreloj de 200m

    Esfuerzo máximo desde salida de pared. Registra el tiempo con precisión. Esto debería ser más rápido por 100m que los 400m.

⚠️ Errores Comunes

Récupération Insuficiente

Problema: La fatigue ralentiza artificialmente el tiempo de 200m

Resultado: El CSS calculado se vuelve más rápido que la realidad, llevando a zones sobreentrenadas

Solución: Descansa hasta que la FC baje de 120 ppm o hasta que la respiración se haya recuperado completamente

Mal Allure en los 400m

Problema: Empezar demasiado rápido causa una desaceleración dramática

Resultado: El tiempo de 400m no refleja el verdadero allure sostenible

Solución: Apunta a splits parejos o split negativo (segundos 200m ≤ primeros 200m)

Usar Salidas de Clavado

Problema: Añade una ventaja de ~0.5-1.5 segundos, sesgando los cálculos

Solución: Siempre usa salida de pared

🔄 Frecuencia de Re-evaluación

Vuelve a evaluar el CSS cada 6-8 semanas para actualizar las zones de entraînement a medida que mejora la condición física. Tus zones deberían volverse progresivamente más rápidas a medida que te adaptas al entraînement.

Fórmula de Calcul del CSS

Fórmula

CSS (m/s) = (D₂ - D₁) / (T₂ - T₁)

Donde:

  • D₁ = 200 metros
  • D₂ = 400 metros
  • T₁ = Tiempo para 200m (en segundos)
  • T₂ = Tiempo para 400m (en segundos)

Simplificado para Allure por 100m

Allure CSS/100m (segundos) = (T₄₀₀ - T₂₀₀) / 2

Ejemplo Trabajado

Resultados de la Test:

  • Tiempo 400m: 6:08 (368 segundos)
  • Tiempo 200m: 2:30 (150 segundos)

Paso 1: Calcular CSS en m/s

CSS = (400 - 200) / (368 - 150)
CSS = 200 / 218
CSS = 0.917 m/s

Paso 2: Convertir a allure por 100m

Allure = 100 / 0.917
Allure = 109 segundos
Allure = 1:49 por 100m

🎯 Beneficios del Entraînement por Zones

Usar zones basadas en CSS transforma el entraînement subjetivo de "sensación" en entraînements objetivos y repetibles. Cada zone se dirige a adaptaciones fisiológicas específicas:

  • Zone 2: Construir motor aeróbico (60-70% del volumen semanal)
  • Zone 3: Mejorar la efficacité al allure de carrera (15-20% del volumen)
  • Zone 4: Elevar el seuil de lactato (10-15% del volumen)
  • Zone 5: Desarrollar vitesse máxima y potencia (5-10% del volumen)

Validación Científica

Wakayoshi et al. (1992-1993) - Investigación Fundacional

Los estudios seminales de Kohji Wakayoshi en la Universidad de Osaka establecieron el CSS como una alternativa válida y práctica a las pruebas de lactato en laboratorio:

  • Fuerte correlación con VO₂ en el seuil anaeróbico (r = 0.818)
  • Excelente correlación con vitesse en OBLA (r = 0.949)
  • Predice el performance en 400m (r = 0.864)
  • Corresponde a 4 mmol/L de lactato en sangre - estado máximo estable de lactato
  • Relación lineal entre distancia y tiempo (r² > 0.998)

Artículos Clave:

  1. Wakayoshi K, et al. (1992). "Determination and validity of critical velocity as an index of swimming performance in the competitive swimmer." European Journal of Applied Physiology, 64(2), 153-157.
  2. Wakayoshi K, et al. (1992). "A simple method for determining critical speed as swimming fatigue threshold in competitive swimming." International Journal of Sports Medicine, 13(5), 367-371.
  3. Wakayoshi K, et al. (1993). "Does critical swimming velocity represent exercise intensity at maximal lactate steady state?" European Journal of Applied Physiology, 66(1), 90-95.

🔬 Por Qué Funciona el CSS

El CSS representa el límite entre los dominios de exercice intenso y severo. Por debajo del CSS, la producción y eliminación de lactato permanecen equilibradas—puedes nadar durante períodos prolongados. Por encima del CSS, el lactato se acumula progresivamente hasta el agotamiento en 20-40 minutos.

Esto hace que el CSS sea la intensidad perfecta para:

  • Establecer ritmos de carrera sostenibles para eventos de 800m-1500m
  • Prescribir entraînement de intervalos de seuil
  • Monitorear mejoras en la condición aeróbica
  • Calcular la charge de entraînement y las necesidades de récupération

Aplicaciones Prácticas

1️⃣ Desbloquear Métricas de Charge de Entraînement

El CSS es el denominador en el calcul del Factor de Intensidad para sTSS. Sin él, no puedes cuantificar el estrés del entraînement ni seguir las tendencias de condición física/fatigue.

2️⃣ Personalizar Zones de Entraînement

Las tablas de allure genéricas no tienen en cuenta la fisiología individual. Las zones basadas en CSS aseguran que cada nageur entrene a su intensidad óptima.

3️⃣ Monitorear la Progresión de la Condición Física

Vuelve a evaluar cada 6-8 semanas. Mejorar el CSS (allure más rápido) indica una adaptación aeróbica exitosa. Un CSS estancado sugiere que el entraînement necesita ajustes.

4️⃣ Predecir el Performance en Carrera

El allure CSS aproxima tu allure de carrera sostenible de 30 minutos. Úsalo para establecer objetivos realistas para eventos de 800m, 1500m y aguas abiertas.

5️⃣ Diseñar Entrenamientos de Seuil

Series clásicas de CSS: 8×100 @ allure CSS (15s descanso), 5×200 @ 101% CSS (20s descanso), 3×400 @ 103% CSS (30s descanso). Construir capacidad de eliminación de lactato.

6️⃣ Optimizar la Estrategia de Descarga

Sigue el CSS antes y después de la descarga. Una descarga exitosa mantiene o mejora ligeramente el CSS mientras reduce la fatigue (TSB aumentado).

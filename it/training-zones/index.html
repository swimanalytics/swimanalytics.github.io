Zone de Allenamento de Nuoto - Guía de Intensidad Basada en CSS

Domina las 5 zone de allenamento para nuoto - Personalizadas a tu Velocità Crítica de Nuoto

¿Qué Son las Zone de Allenamento de Nuoto?

Las zone de allenamento son rangos de intensidad científicamente definidos basados en tu Velocità Crítica de Nuoto (CSS)—tu andatura de soglia aeróbico. Cada zona desencadena adaptaciones fisiológicas específicas, desde la construcción de base aeróbica (Zona 2) hasta el desarrollo del VO₂max (Zona 5). Las zone de allenamento eliminan las conjeturas y aseguran que cada allenamento tenga un propósito claro.

Por Qué Funciona el Allenamento por Zone

Entrenar por "sensación" o tablas de andatura genéricas falla porque:

  • La fisiología individual varía: Un andatura de 1:40/100m es fácil para nuotatori de élite pero máximo para principiantes
  • El RPE no es fiable: El esfuerzo percibido cambia con la fatica, hidratación y condiciones
  • Los ritmos genéricos fallan tu soglia: Los allenamenti estándar ignoran tu soglia de lactato único
  • Sin especificidad de adaptación: Ritmos aleatorios producen resultados aleatorios

Las zone basadas en CSS resuelven esto personalizando cada intensidad a TU fisiología. Un allenamento de Zona 2 desencadena adaptaciones aeróbicas tanto si tu CSS es 1:20/100m como 2:00/100m.

🎯 Principio Clave: Relación Inversa

En nuoto, el andatura se mide como tiempo por distancia. Por lo tanto:

  • Mayor % de CSS = andatura MÁS LENTO (más fácil, Zona 1-2)
  • Menor % de CSS = andatura MÁS RÁPIDO (más difícil, Zona 4-5)

Esto es inverso al ciclismo/carrera donde mayor % = más difícil. Piensa: "andatura 108% CSS" = 8% más lento que el soglia.

Las 5 Zone de Allenamento de Nuoto

Zona Nombre % de Andatura CSS Ejemplo para CSS 1:40/100m RPE Propósito Fisiológico
1 Recupero >108% >1:48/100m 2-3/10 Recupero activa, refinamiento técnico, calentamiento/enfriamiento
2 Base Aeróbica 104-108% 1:44-1:48/100m 4-5/10 Construir capacidad aeróbica, densidad mitocondrial, oxidación de grasas
3 Tempo/Punto Óptimo 99-103% 1:39-1:43/100m 6-7/10 Adaptación al andatura de carrera, efficienza neuromuscular
4 Soglia (CSS) 96-100% 1:36-1:40/100m 7-8/10 Mejora del soglia de lactato, intensidad alta sostenida
5 VO₂max/Anaeróbico <96% <1:36/100m 9-10/10 Desarrollo de VO₂max, potencia, tolerancia al lactato

Zona 1: Recupero

Zona 1 >108% de andatura CSS RPE 2-3/10

Propósito

Recupero activa, trabajo técnico, calentamiento, enfriamiento. La Zona 1 promueve el flujo sanguíneo para la reparación muscular sin crear estrés de allenamento adicional. No se usa para ganancias de condición física—puramente regenerativa.

Marcadores Fisiológicos

  • Frecuencia cardíaca: 50-60% del máximo
  • Lactato: <1.5 mmol/L (muy por debajo del soglia)
  • Respiración: Respiración nasal posible, andatura conversacional
  • Sensación: Sin esfuerzo, podrías mantenerlo indefinidamente

Entrenamientos de Ejemplo

Sesión de Recupero

  • 500m nado continuo @ Zona 1 (enfoque: bracciata suave)
  • 10×25 ejercicios técnicos (alcance, remo, un solo brazo) @ esfuerzo Zona 1
  • 300m pull con pull buoy @ Zona 1

Volumen Semanal

10-20% del volumen total (calentamientos, enfriamientos, nados de recupero en días libres)

Zona 2: Base Aeróbica

Zona 2 104-108% de andatura CSS RPE 4-5/10

Propósito

Fundamento de todo allenamento de resistencia. La Zona 2 construye densidad mitocondrial, redes capilares, capacidad de oxidación de grasas y enzimas aeróbicas. Aquí es donde se construye la condición aeróbica—la zona "aburrida" que produce campeones.

Marcadores Fisiológicos

  • Frecuencia cardíaca: 60-75% del máximo
  • Lactato: 1.5-2.5 mmol/L (por debajo del primer soglia de lactato)
  • Respiración: Rítmica, cómoda, puedes hablar en frases
  • Sensación: Cómodo, sostenible durante 60+ minutos

Entrenamientos de Ejemplo

Sesión de Resistencia Aeróbica

  • 3000m continuo @ andatura Zona 2
  • 20×100 @ andatura Zona 2 (10s descanso)
  • 5×400 @ andatura Zona 2 (20s descanso)

Volumen Semanal

60-70% del volumen total (zona más crítica para el desarrollo de condición física)

⚠️ Error Común: Entrenar Demasiado Fuerte

La mayoría de los nuotatori entrenan la Zona 2 demasiado rápido, empujando hacia la Zona 3-4. Esto crea fatica crónica sin construir base aeróbica. La Zona 2 debe sentirse fácil—deberías terminar sintiendo que podrías hacer más.

Zona 3: Tempo / Punto Óptimo

Zona 3 99-103% de andatura CSS RPE 6-7/10

Propósito

Adaptación al andatura de carrera para pruebas de media distancia (400m-1500m). La Zona 3 entrena la efficienza neuromuscular a velocidades de carrera sostenibles. También conocida como allenamento de "Punto Óptimo"—más difícil que la base, más fácil que el soglia, con buenas adaptaciones aeróbicas por unidad de fatica.

Marcadores Fisiológicos

  • Frecuencia cardíaca: 75-85% del máximo
  • Lactato: 2.5-4.0 mmol/L (acercándose al soglia)
  • Respiración: Controlada pero elevada, solo frases cortas
  • Sensación: Cómodamente difícil, sostenible durante 20-40 minutos

Entrenamientos de Ejemplo

Sesión de Tempo

  • 10×200 @ andatura Zona 3 (15s descanso)
  • 3×800 @ andatura Zona 3 (30s descanso)
  • 2000m fraccionado (500-400-300-400-500) @ andatura Zona 3 (20s descanso entre series)

Volumen Semanal

15-20% del volumen total (clave para preparación específica de carrera)

Zona 4: Soglia (Andatura CSS)

Zona 4 96-100% de andatura CSS RPE 7-8/10

Propósito

Allenamento de soglia de lactato—la "zona valiosa". La Zona 4 eleva tu soglia anaeróbico, mejorando tu capacidad para eliminar lactato y sostener esfuerzos de alta intensidad. Este es tu andatura CSS—la velocità más rápida que puedes mantener durante ~30 minutos sin agotamiento.

Marcadores Fisiológicos

  • Frecuencia cardíaca: 85-92% del máximo
  • Lactato: 4.0-6.0 mmol/L (estado estable máximo de lactato)
  • Respiración: Dura, laboriosa, solo palabras sueltas
  • Sensación: Muy duro, sostenible durante 20-30 minutos máximo

Entrenamientos de Ejemplo

Sesión de Soglia

  • 8×100 @ andatura CSS (15s descanso) — serie clásica CSS
  • 5×200 @ 101% CSS (20s descanso)
  • 3×400 @ 103% CSS (30s descanso)
  • 1500m contrarreloj continuo @ andatura CSS

Volumen Semanal

10-15% del volumen total (alto estrés de allenamento, requiere recupero adecuada)

💡 Consejo Pro: Usa sTSS para Gestionar la Carico de Zona 4

Los allenamenti de Zona 4 generan 150-250 sTSS por sesión. Rastrea los totales semanales para evitar el sobreentrenamiento. Limita el trabajo de Zona 4 a 2-3 sesiones por semana máximo durante las fases de construcción.

Zona 5: VO₂max / Anaeróbico

Zona 5 <96% de andatura CSS (más rápido que CSS) RPE 9-10/10

Propósito

Desarrollo de VO₂max, capacidad anaeróbica, tolerancia al lactato y potencia neuromuscular. La Zona 5 entrena tu cuerpo para producir y tolerar altos niveles de lactato. Se usa para pruebas de sprint (50m-200m) y desarrollo de velocità máxima.

Marcadores Fisiológicos

  • Frecuencia cardíaca: 92-100% del máximo
  • Lactato: 6.0-15+ mmol/L (acumulación severa)
  • Respiración: Máxima, jadeante, sin conversación posible
  • Sensación: Esfuerzo total, sostenible solo durante 2-8 minutos

Entrenamientos de Ejemplo

Sesión de VO₂max

  • 12×50 @ esfuerzo máximo (30s descanso)
  • 6×100 @ andatura carrera 200m (60s descanso)
  • 4×200 @ andatura 94% CSS (90s descanso)
  • 20×25 sprint total (15s descanso)

Volumen Semanal

5-10% del volumen total (coste de fatica más alto, usar con moderación)

⚠️ Recupero Crítica

El trabajo de Zona 5 es extremadamente exigente. Requiere 48-72 horas de recupero entre sesiones. NO acumules allenamenti de Zona 5 en días consecutivos. Rastrea CTL/ATL/TSB para asegurar recupero adecuada.

Distribución Semanal de Allenamento por Nivel de Atleta

Nadadores Recreativos / Fitness

Volumen Total: 6,000-12,000m/semana (2-3 sesiones)

  • Zona 1: 15% (calentamiento/enfriamiento)
  • Zona 2: 70% (construir base aeróbica)
  • Zona 3: 10% (tempo ocasional)
  • Zona 4: 5% (trabajo de soglia limitado)
  • Zona 5: 0% (no necesario todavía)

Nadadores Masters Competitivos

Volumen Total: 15,000-25,000m/semana (4-6 sesiones)

  • Zona 1: 15% (nados de recupero)
  • Zona 2: 60% (fundamento aeróbico)
  • Zona 3: 15% (trabajo de andatura de carrera)
  • Zona 4: 8% (sesiones de soglia)
  • Zona 5: 2% (desarrollo de velocità)

Triatletas (Enfoque Nuoto)

Volumen Total: 10,000-18,000m/semana (3-4 sesiones)

  • Zona 1: 10% (calentamiento/tecnica)
  • Zona 2: 75% (maximizar efficienza aeróbica)
  • Zona 3: 10% (simulación de carrera)
  • Zona 4: 5% (limitado—preservar energía para ciclismo/carrera)
  • Zona 5: 0% (no relevante para carreras de resistencia)

Nadadores Élite / Universitarios

Volumen Total: 40,000-70,000m/semana (10-12 sesiones)

  • Zona 1: 20% (recupero esencial con alto volumen)
  • Zona 2: 50% (mantenimiento de base aeróbica)
  • Zona 3: 15% (especificidad de andatura de carrera)
  • Zona 4: 10% (desarrollo de soglia)
  • Zona 5: 5% (potencia y velocità)

Cómo Calcular Tus Zone de Allenamento Personales

Tus zone de allenamento están personalizadas a TU Velocità Crítica de Nuoto. Aquí está cómo calcularlas:

Paso 1: Realizar Test CSS

Completa un contrarreloj de 400m y 200m con 5-10 minutos de recupero entre esfuerzos. Aprende el protocolo completo del test CSS →

Paso 2: Calcular Andatura CSS

Ejemplo:

  • Tiempo 400m: 6:08 (368 segundos)
  • Tiempo 200m: 2:30 (150 segundos)

Andatura CSS = (T₄₀₀ - T₂₀₀) / 2
Andatura CSS = (368 - 150) / 2 = 109 segundos = 1:49/100m

Paso 3: Calcular Ritmos de Zona

Multiplica el andatura CSS por los porcentajes de zona:

Zona Rango % Calcolo (CSS = 1:49/100m) Rango de Andatura de Zona
Zona 1 >108% 109 × 1.08 = 118s >1:58/100m
Zona 2 104-108% 109 × 1.04-1.08 = 113-118s 1:53-1:58/100m
Zona 3 99-103% 109 × 0.99-1.03 = 108-112s 1:48-1:52/100m
Zona 4 96-100% 109 × 0.96-1.00 = 105-109s 1:45-1:49/100m
Zona 5 <96% 109 × 0.96 = 105s <1:45/100m

Preguntas Frecuentes sobre Zone de Allenamento

¿Con qué frecuencia debo volver a testar mi CSS para actualizar las zone?

Cada 6-8 semanas durante las fases de base y construcción. Tu CSS debería mejorar (hacerse más rápido) a medida que aumenta la condición física, requiriendo ajustes de zona. Vuelve a testar después de enfermedad, lesión o descansos largos.

¿Puedo mezclar zone en un solo allenamento?

Sí—la mayoría de los allenamenti son multizona. Ejemplo: 400m calentamiento Zona 1 + 8×100 soglia Zona 4 + 300m enfriamiento Zona 1. La clave es la selección intencional de zona, no nadar accidentalmente en "zona media".

¿Qué pasa si no puedo mantener el andatura de zona?

Si no puedes mantener el andatura de zona prescrito, entonces: (1) tu CSS está desactualizado (demasiado rápido), (2) estás fatigado (revisa TSB), o (3) recupero insuficiente entre intervalos. Vuelve a testar CSS si esto ocurre consistentemente.

¿Las zone se aplican a todos los estilos?

El CSS se testea típicamente en estilo libre. Para otros estilos, puedes realizar tests CSS específicos del estilo, pero la mayoría de los nuotatori usan las zone CSS de estilo libre y ajustan según la sensación para estilos/espalda/braza.

¿Cómo se relacionan las zone con el Training Stress Score (sTSS)?

La zona determina el Factor de Intensidad (IF), que se eleva al cuadrado en la fórmula sTSS. Zona 4 (IF ~0.95-1.0) genera 90-100 sTSS por hora. Zona 2 (IF ~0.80) genera solo 64 sTSS por hora. Zone más altas = estrés de allenamento exponencialmente mayor.

¿Puedo entrenar solo en Zona 2?

El allenamento solo en Zona 2 funciona para principiantes construyendo condición física base. Sin embargo, los nuotatori avanzados necesitan trabajo de Zona 3-5 para desarrollar adaptaciones específicas de carrera. Sigue la regla 80/20: 80% fácil (Zona 1-2), 20% difícil (Zona 3-5).

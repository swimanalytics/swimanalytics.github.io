¿Qué es TSS Natación (sTSS)?

Swimming Training Stress Score (sTSS) cuantifica beban latihan sesión renang combinando intensidad dan duración. Está adaptado metodología TSS ciclismo, utilizando Anda Critical Swim Speed (CSS) sebagai ritmo umbral. Un latihan 1 hora ritmo CSS = 100 sTSS.

Interpretación Beban Latihan

Cómo se Calcula sTSS

Fórmula

sTSS = (Duración di horas) × (Factor Intensidad)² × 100

Donde:

  • Factor Intensidad (IF) = Ritmo CSS / Ritmo Promedio Latihan
  • Duración = Waktu total latihan di horas
  • Ritmo CSS = Anda ritmo umbral test CSS

Ejemplo Práctico

Detalles Latihan:

  • Ritmo CSS: 1:49/100m (109 segundos)
  • Duración Latihan: 60 minutos (1 hora)
  • Ritmo Promedio: 2:05/100m (125 segundos)

Paso 1: Calcular Factor Intensidad

IF = Ritmo CSS / Ritmo Latihan
IF = 109 / 125
IF = 0.872

Paso 2: Calcular sTSS

sTSS = 1.0 horas × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Interpretación: Este latihan 60 minutos ritmo suave (lebih dari lento yang CSS) generó 76 sTSS - beban latihan moderada adecuada untuk construir base aeróbica.

Guía sTSS Semanal

El sTSS semanal objetivo depende Anda nivel latihan dan objetivos:

Nadadores Recreativos

sTSS Semanal: 150-300

2-3 latihans oleh semana, 50-100 sTSS setiap uno. Enfoyang di técnica dan construcción base aeróbica.

Nadadores Fitness / Triatlet

sTSS Semanal: 300-500

3-4 latihans oleh semana, 75-125 sTSS setiap uno. Mezcla resistencia aeróbica dan trabajo umbral.

Nadadores Másters Competitivos

sTSS Semanal: 500-800

4-6 latihans oleh semana, 80-150 sTSS setiap uno. Latihan estructurado dengan periodización.

Nadadores Elite / Universitarios

sTSS Semanal: 800-1200+

8-12 latihans oleh semana, sesi dobles. Alto volumen dengan manajemen recuperación kritis.

⚠️ Notas Importantes

  • Requiere CSS preciso: Anda CSS debe estar actualizado (testeado dentro 6-8 semanas) untuk sTSS preciso.
  • Cálculo simplificado: Esta calculadora usa ritmo promedio. El sTSS avanzado usa Normalized Graded Pace (NGP) yang tiene di cuenta estructura intervalos.
  • No untuk trabajo técnico: El sTSS solo mide stres físico latihan, tidak pengembangan habilidades.
  • Variación individual: El sama sTSS se siente diferente untuk diferentes nadadores. Ajusta guías según Anda recuperación.

Por Qué Importa sTSS

El Training Stress Score es base para:

  • CTL (Chronic Training Load): Anda nivel forma física - promedio ponderado exponencialmente 42 días sTSS diario
  • ATL (Acute Training Load): Anda fatiga - promedio ponderado exponencialmente 7 días sTSS diario
  • TSB (Training Stress Keseimbangan): Anda estado forma - TSB = CTL - ATL (positivo = fresco, negativo = fatigado)
  • Periodización: Planificar fases latihan (base, pengembangan, pico, descenso) usando progresiones CTL objetivo
  • Manajemen Recuperación: Saber cuándo empujar dan cuándo descansar según TSB

Consejo Pro: Sigue Anda CTL

Registra sTSS diario di hoja cálculo atau diario latihan. Calcula Anda promedio 42 días (CTL) semanalmente. Objetivo aumento 5-10 puntos CTL oleh semana durante construcción base. Mantén atau reduce ligeramente CTL durante descenso (1-2 semanas antes competición).

Swimming Training Stress Score (sTSS)

Wat het is: Gekwantificeerde trainingsstress die intensiteit en duur combineert. 1 uur op CSS tempo = 100 sTSS.

Wat het betekent: sTSS 50 = makkelijk herstel, sTSS 100 = gematigd, sTSS 200+ = zeer zware sessie.

Hoe te gebruiken: Volg dagelijkse/wekelijkse sTSS om trainingsbelasting te beheren. Streef naar max 5-10 sTSS per week stijging.

sTSS Gids ‚Üí

CTL / ATL / TSB

Wat ze zijn:

  • CTL: Chronic Training Load (fitness) - 42-daags gemiddelde van sTSS
  • ATL: Acute Training Load (vermoeidheid) - 7-daags gemiddelde van sTSS
  • TSB: Training Stress Balance (vorm) = CTL - ATL

Hoe te gebruiken: TSB positief = fris/taper, TSB negatief = vermoeid. Wedstrijd wanneer TSB = +5 tot +25.

Probleemoplossing en Veelgestelde Vragen

Mijn sTSS lijkt te hoog/laag voor de trainingsinspanning

Oorzaak: CSS is verouderd of onnauwkeurig.

Oplossing: Hertest CSS. Als je testte toen je vermoeid was of verkeerd doseerde, zal CSS incorrect zijn. Een goede CSS test is cruciaal voor alle opvolgende metrics.

De app toont "Geen CSS ingesteld"

Oorzaak: CSS test niet voltooid of niet opgeslagen.

Oplossing: Ga naar Instellingen ‚Üí Critical Swim Speed ‚Üí Test Doen. Voer beide 400m en 200m tijden in, tik dan op Opslaan.

Trainingen synchroniseren niet vanaf Apple Watch

Oorzaak: Health app permissies niet verleend of training niet gecategoriseerd als "Zwemmen".

Oplossing: Verifieer Instellingen ‚Üí Privacy ‚Üí Gezondheid ‚Üí SwimAnalytics ‚Üí Sta Trainingen Lezen toe. Zorg dat trainingstype op Apple Watch "Zwembad Zwemmen" of "Open Water Zwemmen" is.

CTL stijgt niet ondanks consistente training

Oorzaak: sTSS totalen te laag of inconsistente frequentie.

Oplossing: CTL is een exponentieel gewogen 42-daags gemiddelde. Stijgt langzaam. Verhoog wekelijkse sTSS met 5-10%, en handhaaf 4+ trainingen/week voor consistente CTL groei.

Hoe vaak moet ik CSS hertesten?

Aanbeveling: Elke 6-8 weken tijdens basis/opbouw fasen. Hertest na ziekte, blessure, lange pauze, of wanneer zones consistent te makkelijk/moeilijk aanvoelen.

Kan ik SwimAnalytics gebruiken voor andere slagen?

Ja, met beperkingen: CSS wordt typisch getest op vrije slag. Voor wisselslag/rugslag/schoolslag trainingen wordt sTSS berekend op basis van vrije slag CSS. Overweeg slag-specifieke CSS tests voor betere nauwkeurigheid.

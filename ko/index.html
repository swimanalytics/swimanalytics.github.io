수영 훈련을 위한 과학 기반 분석

CSS(Critical Swim Speed) 기반 훈련 구역을 통해 수영 성과를 향상시키고 sTSS, CTL, ATL 및 TSB로 훈련 부하를 추적하세요. 모든 수영 선수를 위한 전문적인 분석 도구입니다.

주요 기능

CSS 기반 훈련 구역

Critical Swim Speed를 계산하고 과학적으로 검증된 훈련 구역에서 훈련하세요. CSS 테스트(400m + 200m)는 간단하면서도 강력한 유산소 역치 측정 방법입니다.

CSS에 대해 알아보기

sTSS 훈련 부하

수영 Training Stress Score로 매 훈련 세션의 강도를 정량화하세요. CSS 기반 구역을 사용하여 계산되어 수영에 맞게 조정된 정확한 부하 측정을 제공합니다.

sTSS 계산기

CTL/ATL/TSB 추적

Performance Management Chart로 체력, 피로도 및 회복을 모니터링하세요. 최적의 타이밍에 최고의 경기력을 발휘하고 과훈련을 예방하세요.

훈련 부하 알아보기

스트로크 역학

Stroke Rate(SR), Distance Per Stroke(DPS) 및 Stroke Index(SI)를 추적하여 기술 효율성을 최적화하세요. 속도 = SR × DPS를 이해하여 더 스마트하게 수영하세요.

스트로크 역학 알아보기

SWOLF 효율성

SWOLF(Swim Golf)로 수영 효율성을 측정하세요: 스트로크 수 + 시간(초). 낮을수록 좋습니다. 거리별로 SWOLF를 추적하고 수영 경제성을 향상시키세요.

SWOLF 알아보기

진행 상황 추적

시간 경과에 따른 CSS 향상, 구역 페이스 및 SWOLF 트렌드를 시각화하세요. 데이터 기반 인사이트로 훈련이 효과를 내고 있는지 확인하세요.

시작하는 방법

SwimAnalytics를 선택하는 이유

수영에 특화된 설계

다른 앱들과 달리 SwimAnalytics는 처음부터 수영 선수를 위해 설계되었습니다. CSS는 러닝의 FTP에 해당하는 것으로, 훈련 구역과 부하 계산을 위한 완벽한 기준을 제공합니다.

과학적 근거

저희 방법론은 Wakayoshi et al. (1992)의 CSS 프로토콜과 Coggan의 TSS 프레임워크를 기반으로 합니다. 추측이 아닌 검증된 스포츠 과학입니다.

심플하면서도 강력함

복잡한 장비가 필요 없습니다. 400m + 200m 타임 트라이얼만으로 CSS를 계산하고 정확한 훈련 구역을 얻을 수 있습니다. 설정 시간은 10분이며 평생 사용할 수 있습니다.

Strava와 TrainingPeaks는 어떤가요?

Strava는 러닝용으로 훌륭하지만 수영 분석에는 기본적인 기능만 제공합니다. TrainingPeaks는 강력하지만 월 $20이며 CSS 구역을 수동으로 설정해야 합니다.

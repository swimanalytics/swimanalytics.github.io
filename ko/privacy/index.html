개인정보 보호정책

최종 수정일: 2025년 1월 8일

SwimAnalytics("저희", "우리" 또는 "앱")는 귀하의 개인정보 보호를 존중합니다. 본 개인정보 보호정책은 SwimAnalytics iOS 앱을 사용할 때 저희가 귀하의 정보를 수집, 사용, 공개 및 보호하는 방법을 설명합니다.

1. 저희가 수집하는 정보

1.1. 귀하가 제공하는 정보

SwimAnalytics를 사용할 때 다음 정보를 제공할 수 있습니다:

  • 수영 데이터: CSS 테스트 결과(400m 및 200m 시간), 훈련 세션(거리, 페이스, 기간), 스트로크 역학(SR, DPS, SI), SWOLF 점수
  • 계정 정보: Apple을 통해 로그인하는 경우 이메일 주소(선택 사항)
  • 사용자 설정: 훈련 구역 사용자 정의, 단위 기본 설정(미터/야드), 앱 설정

1.2. 자동으로 수집되는 정보

저희는 익명화된 사용 데이터를 수집하여 앱을 개선할 수 있습니다:

  • 사용 통계: 기능 사용 빈도, 앱 충돌 보고서, 성능 메트릭
  • 기기 정보: iOS 버전, 기기 모델, 앱 버전
  • 분석 데이터: 익명화된 사용 패턴, 기능 채택률

1.3. HealthKit 통합

명시적 권한을 부여하면 SwimAnalytics는 Apple HealthKit에서 다음을 읽고 쓸 수 있습니다:

  • 읽기: 수영 거리, 기간, 에너지 소모(선택 사항)
  • 쓰기: 계산된 sTSS를 "활동 에너지"로 기록(선택 사항)

중요: HealthKit 데이터는 절대 저희 서버로 전송되지 않습니다. 귀하의 기기에 로컬로 남아 있으며 Apple의 건강 데이터 개인정보 보호 정책이 적용됩니다.

2. 저희가 귀하의 정보를 사용하는 방법

저희는 수집된 정보를 다음 용도로 사용합니다:

  • 서비스 제공: CSS 계산, 훈련 구역, sTSS, CTL/ATL/TSB 차트 및 스트로크 분석
  • 데이터 저장: 귀하의 수영 기록, 진행 상황 및 설정을 로컬로 저장
  • 앱 개선: 익명화된 사용 데이터를 분석하여 기능 및 성능 개선
  • 고객 지원: 질문에 답변하고, 문제 해결 및 지원 제공
  • 법적 준수: 법적 의무 준수 및 저희 이용 약관 집행

3. 데이터 저장 및 보안

3.1. 로컬 저장

모든 수영 데이터(운동, CSS 테스트, 스트로크 메트릭)는 귀하의 iPhone에 로컬로 저장됩니다:

  • 암호화된 CoreData 데이터베이스에 저장
  • iCloud(활성화된 경우) 또는 iTunes를 통해 백업
  • 귀하가 기기를 제어하므로 귀하만 액세스 가능

3.2. 클라우드 저장소 없음

SwimAnalytics는 귀하의 운동 데이터를 저희 서버로 업로드하지 않습니다. 귀하의 수영 기록, CSS 값 및 훈련 세션은 귀하의 것이며 귀하의 기기에 남아 있습니다. 저희는 귀하의 개인 피트니스 데이터를 수집, 저장 또는 공유하지 않습니다.

3.3. 보안 조치

저희는 산업 표준 보안 관행을 구현합니다:

  • 저장 시 데이터 암호화 (iOS 파일 시스템 암호화)
  • App Store를 통한 안전한 인증 (Apple로 로그인)
  • 정기적인 보안 감사 및 업데이트
  • 최소 권한 액세스 원칙

4. 데이터 공유 및 공개

4.1. 제3자와 공유 안 함

저희는 귀하의 개인 수영 데이터(운동, CSS 테스트, 메트릭)를 제3자에게 판매, 임대 또는 공유하지 않습니다. 귀하의 데이터는 귀하의 것입니다.

4.2. 제3자 서비스

SwimAnalytics는 다음 제3자 서비스를 사용합니다:

  • Apple App Store: 구독 결제 처리용(Apple의 개인정보 보호정책 적용)
  • Apple HealthKit: 건강 데이터 통합용(Apple의 건강 개인정보 보호정책 적용)
  • 분석 제공업체: 익명화된 앱 사용 통계용(Google Analytics 또는 유사)

4.3. 법적 요구사항

저희는 다음과 같은 경우 정보를 공개할 수 있습니다:

  • 법적 절차, 소환장 또는 법원 명령에 의해 요구되는 경우
  • 저희의 권리나 재산을 보호하기 위해 필요한 경우
  • 사기 또는 보안 문제를 조사하기 위해

5. 귀하의 권리 및 선택

5.1. 데이터 액세스 및 내보내기

귀하는 언제든지 다음을 할 수 있습니다:

  • 앱 내 "데이터 내보내기" 기능을 통해 귀하의 모든 데이터를 CSV 형식으로 내보내기
  • 저장된 모든 운동, CSS 테스트 및 메트릭 보기
  • 지원팀에 연락하여 데이터 사본 요청

5.2. 데이터 삭제

귀하는 다음을 통해 귀하의 데이터를 삭제할 수 있습니다:

  • 앱 설정에서 개별 운동 또는 CSS 테스트 삭제
  • 앱 설정의 "모든 데이터 삭제" 옵션 사용
  • iPhone에서 SwimAnalytics 앱 제거(모든 로컬 데이터 삭제)

5.3. HealthKit 권한

다음을 통해 HealthKit 권한을 관리할 수 있습니다:

  • iOS 설정 → 건강 → 데이터 액세스 및 기기 → SwimAnalytics
  • 언제든지 읽기 또는 쓰기 권한 취소

5.4. 분석 선택 해제

앱 설정에서 익명화된 사용 통계 공유를 선택 해제할 수 있습니다. 이는 핵심 앱 기능에 영향을 미치지 않습니다.

6. 어린이 개인정보 보호

SwimAnalytics는 만 13세 미만의 어린이를 대상으로 하지 않습니다. 저희는 만 13세 미만 어린이로부터 의도적으로 개인정보를 수집하지 않습니다. 만 13세 미만 아동이 데이터를 제공했다고 생각되면 privacy@swimanalytics.app로 문의해 주시면 즉시 삭제하겠습니다.

7. 국제 데이터 전송

귀하의 수영 데이터는 귀하의 기기에 로컬로 저장되므로 국제 전송이 없습니다. 지원 문의는 유럽 연합(스페인)에 있는 저희 서버에서 처리될 수 있습니다. 익명화된 분석 데이터는 제3자 제공업체의 개인정보 보호정책에 따라 처리될 수 있습니다.

8. 데이터 보존

귀하의 수영 데이터는 귀하가 삭제하거나 앱을 제거할 때까지 귀하의 기기에 남아 있습니다. 저희는 다음을 보존합니다:

  • 수영 데이터: 귀하가 삭제할 때까지(로컬 저장소)
  • 지원 이메일: 문제 해결 후 2년
  • 익명화된 분석: 집계 형식으로 무기한

9. 캘리포니아 개인정보 보호 권리 (CCPA)

캘리포니아 거주자인 경우 다음 권리가 있습니다:

  • 알 권리: 저희가 수집하는 개인정보 카테고리 요청
  • 삭제 권리: 귀하의 개인정보 삭제 요청
  • 판매 선택 해제: 저희는 개인정보를 판매하지 않습니다
  • 차별 금지: 이러한 권리 행사에 대한 차별 없음

10. 유럽 사용자 (GDPR)

유럽 경제 지역(EEA) 또는 영국 거주자인 경우 다음 권리가 있습니다:

  • 액세스: 저희가 보유한 귀하의 데이터 사본 받기
  • 수정: 부정확한 개인정보 수정
  • 삭제: 귀하의 개인정보 삭제 요청("잊힐 권리")
  • 이식성: 구조화된 형식으로 데이터 받기
  • 반대: 특정 처리 활동에 반대
  • 철회: 언제든지 동의 철회

이러한 권리를 행사하려면 privacy@swimanalytics.app로 문의하십시오.

11. 본 정책 변경

저희는 이 개인정보 보호정책을 수시로 업데이트할 수 있습니다. 변경 사항은 이 페이지에 "최종 수정일" 날짜가 업데이트되면 효력이 발생합니다. 중요한 변경 사항은 앱 내 알림 또는 이메일을 통해 알려드립니다. 업데이트된 정책을 정기적으로 검토하는 것이 좋습니다.

12. 문의하기

이 개인정보 보호정책이나 귀하의 데이터에 대한 질문이 있으시면 다음으로 문의해 주십시오:

이메일: privacy@swimanalytics.app
지원: 문의 페이지
데이터 보호 담당자: 요청 시 이용 가능

요약: 귀하의 개인정보 보호가 중요합니다

  • ✓ 귀하의 운동 데이터는 귀하의 기기에 남아 있습니다(저희 서버에 없음)
  • ✓ 저희는 귀하의 수영 데이터를 판매하거나 공유하지 않습니다
  • ✓ HealthKit 데이터는 절대 저희에게 전송되지 않습니다
  • ✓ 귀하는 언제든지 모든 데이터를 내보내거나 삭제할 수 있습니다
  • ✓ 최소한의 익명화된 분석만 수집됩니다

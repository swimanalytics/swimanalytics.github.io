游泳效率：SWOLF

您的划水经济性得分 - 越低越好

什么是 SWOLF？

SWOLF（游泳 + 高尔夫）是一个综合效率指标，将划水次数和时间合并为一个数字。就像高尔夫一样，目标是使您的得分最小化。

公式

SWOLF = 单程时间（秒）+ 划水次数

示例： 如果您以20秒游25米，划水15次：

SWOLF = 20 + 15 = 35

跨泳池比较的标准化 SWOLF

要在不同泳池长度之间比较得分：

SWOLF₂₅ = (时间 × 25/泳池长度) + (划水次数 × 25/泳池长度)

⚠️ 个体差异

SWOLF 受身高和臂展影响。身材较高的游泳者自然划水次数较少。使用 SWOLF 跟踪您自己的进步，而不是与他人比较。

解读 SWOLF 模式

📉 SWOLF 下降 = 效率提高

您的技术正在改善，或您在给定配速下变得更经济。这是数周和数月训练的目标。

示例： 在8周专注技术训练期间，SWOLF 从48下降至45再到42。

📈 SWOLF 上升 = 效率下降

疲劳正在累积，技术正在恶化，或您游得比效率允许的速度更快。

示例： 在1000米组的最后200米，SWOLF 从42上升至48，表明疲劳。

📊 相同 SWOLF 的不同组合

SWOLF 为45可能来自多种划水/时间组合：

  • 20秒 + 25次划水 = 高频率，较短划水
  • 25秒 + 20次划水 = 低频率，较长划水

始终分析组成部分（划水次数和时间）以了解您的游泳策略。

🎯 SWOLF 训练应用

  • 技术训练： 通过更好的抓水、流线型和身体姿势尝试降低 SWOLF
  • 疲劳监控： SWOLF 上升表明技术恶化 - 该休息了
  • 配速-效率平衡： 找到在 SWOLF 不飙升的情况下能维持的最快配速
  • 练习有效性： 记录练习组前后的 SWOLF 以衡量技术转移

测量最佳实践

📏 划水计数

  • 计算每次入水（两臂合计）
  • 从推壁后第一次划水开始计数
  • 计数直到触壁
  • 保持一致的推壁距离（距旗帜约5米）

⏱️ 计时

  • 从第一次划水测量到触壁
  • 在各趟之间使用一致的推壁强度
  • 技术（Garmin、Apple Watch、FORM）自动计算
  • 手动计时：使用泳池时钟或秒表

🔄 一致性

  • 在相似配速下测量 SWOLF 以进行比较
  • 在主组期间记录，而非热身/放松
  • 注明泳姿类型（自由泳、仰泳等）
  • 比较相同泳池长度（25米对25米，不是25米对50米）

SWOLF 的局限性

🚫 无法跨运动员比较

身高、臂长和灵活性在划水次数上造成自然差异。1.88米的游泳者在相同健康水平下的 SWOLF 会低于1.68米的游泳者。

解决方案： 仅使用 SWOLF 跟踪您的个人进步。

🚫 综合得分隐藏细节

SWOLF 合并了两个变量。您可能改善一个而恶化另一个，仍然获得相同分数。

解决方案： 始终分别检查划水次数和时间。

🚫 未按配速标准化

SWOLF 随着游泳速度加快而自然增加（更多划水，更少时间，但总数更高）。这不是低效率 - 这是物理学。

解决方案： 在特定目标配速下记录 SWOLF（例如，"CSS 配速下的 SWOLF" 对比 "轻松配速下的 SWOLF"）。

🔬 游泳经济性背后的科学

Costill 等人（1985）的研究确立了游泳经济性（每单位距离的能量成本）对中距离表现比 VO₂max 更重要。

SWOLF 作为经济性的粗略指标 - 较低的 SWOLF 通常与给定配速下较低的能量消耗相关，使您能够以相同努力游得更快或更久。

SWOLF 训练练习

🎯 SWOLF 降低组

8 × 50米（30秒休息）

  1. 50 #1-2： 以舒适配速游泳，记录基线 SWOLF
  2. 50 #3-4： 减少2次划水，保持相同时间 → 专注于单次划水距离
  3. 50 #5-6： 略微增加划频，保持相同次数 → 专注于转动
  4. 50 #7-8： 找到最佳平衡 - 寻求最低 SWOLF

目标： 发现您最有效的次数/频率组合。

⚡ SWOLF 稳定性测试

10 × 100米 @ CSS 配速（20秒休息）

记录每个100米的 SWOLF。分析：

  • 哪个100米的 SWOLF 最低？（最高效）
  • SWOLF 在哪里飙升？（技术恶化或疲劳）
  • 从第一个到最后一个100米，SWOLF 变化了多少？

目标： 在所有重复中保持 SWOLF ±2点。一致性表明在疲劳下技术扎实。

效率来自重复

SWOLF 不会一夜之间改善。它是数千次技术正确的划水、刻意练习和有意识关注效率而非速度的累积结果。

持续记录。逐步改善。观察您的游泳如何转变。
